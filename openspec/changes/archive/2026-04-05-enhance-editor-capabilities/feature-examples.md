# 新功能示例

## 1. 多级标题

支持 6 个级别的标题，用于组织剧本结构：

```markdown
# 第一幕

## 场景一：咖啡馆

### 细节说明

#### 角色备注

##### 临时标注

###### 小字说明
```

**工具栏操作：**
- 点击标题下拉菜单（H1-H6）选择级别
- 使用斜杠命令 `/` 输入 "标题"

**快捷键：**
- Ctrl/Cmd + Alt + 1~6

## 2. 引用块

用于引用或强调重要内容：

```
❝ 这是一个引用块，通常用于显示重要的引用文字。
```

**工具栏操作：**
- 点击引用按钮（❝）

**快捷键：**
- Ctrl/Cmd + Shift + B

## 3. 列表

### 无序列表

```
• 角色列表
• 道具清单
• 场景要点
  ○ 子项目
    ▪ 更深层级
```

### 有序列表

```
1. 第一场
2. 第二场
3. 第三场
   1. 子场景
   2. 子场景
```

**工具栏操作：**
- 点击无序列表按钮（•）
- 点击有序列表按钮（1.）

## 4. 代码块

用于插入代码片段或格式化文本：

```
function dialogue() {
  console.log("Hello, World!");
}
```

**工具栏操作：**
- 点击代码块按钮（⌨）

## 5. 水平分割线

用于分隔不同章节：

```
— — — — — — — — — —
```

**工具栏操作：**
- 点击分割线按钮（—）

## 6. 斜杠命令菜单

输入 `/` 打开快速命令菜单，包含：

### 剧本元素
- 🎬 场景 - 插入场景标题
- 💬 对白 - 插入角色对白
- 🎭 动作 - 插入动作描述
- 🔄 转场 - 插入转场效果
- 📝 注释 - 插入备注注释

### 文本格式
- H1-H6 - 插入各级标题
- ❝ 引用 - 插入引用块
- • 无序列表 - 插入无序列表
- 1. 有序列表 - 插入有序列表
- ⌨ 代码块 - 插入代码块
- — 分割线 - 插入水平线

**使用方法：**
1. 输入 `/`
2. 输入关键词搜索（如 "标题"、"引用"）
3. 使用 ↑↓ 箭头导航
4. 按 Enter 选择

## 7. 节点工具栏

鼠标悬停在剧本节点上时，显示工具栏：

- ⬆ 上移 - 将节点向上移动一位
- ⬇ 下移 - 将节点向下移动一位
- 🗑 删除 - 删除当前节点

**特性：**
- 在文档开头时，上移按钮禁用
- 在文档末尾时，下移按钮禁用
- 悬停显示，不干扰编辑

## 8. 节点视图样式

### 场景节点 (Scene)
- 🔵 蓝色系背景
- 🎬 电影图标
- 渐变背景和左边框

### 对白节点 (Dialogue)
- 🟣 紫色系背景
- 💬 对话图标
- 角色名输入框
- 括号说明输入框

### 动作节点 (Action)
- 🟢 绿色系背景
- 🎭 戏剧图标
- 渐变背景和左边框

### 转场节点 (Transition)
- 🟠 橙色系背景
- ➡️ 箭头图标
- 右对齐大写文本

## 9. 完整示例

```
# 我的短剧

## 第一幕

🎬 **场景一：咖啡馆**

💬 **小明**
（走进咖啡馆）
: 老板，来杯咖啡！

🎭 小明走到吧台前坐下，环顾四周。

---

❝ 咖啡馆是一个温馨的地方，充满了咖啡的香气。

### 角色说明

• 小明 - 主角
• 老板 - 配角

🔄 切至下一场景
```

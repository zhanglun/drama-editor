## Context

需要将莎士比亚的《哈姆雷特》完整剧本从文本格式转换为 TipTap JSON 格式。原始文本来源：https://www.gutenberg.org/files/1524/1524-h/1524-h.htm

当前状态：
- 现有 seed 文件仅包含 3 个示例场景
- 需要转换为完整的 5 幕 20 场内容
- 预计文件大小约 100KB

## Goals / Non-Goals

**Goals:**
- 将完整的《哈姆雷特》剧本转换为 TipTap JSON 格式
- 包含所有 5 幕 20 场的完整内容
- 包含所有角色的对白和舞台指示
- 保持 TipTap JSON 格式的一致性

**Non-Goals:**
- 不修改 TipTap 节点类型定义
- 不优化或简化原始剧本内容
- 不添加新的功能特性

## Decisions

**决策 1: 节点类型映射**
- 场景标题 → `scene` 节点
- 角色对白 → `dialogue` 节点（带 `attrs.character` 属性）
- 舞台指示 → `action` 节点
- 转场 → `transition` 节点
- 理由：保持与现有 seed 数据格式一致

**决策 2: 角色列表**
- 包含所有主要角色和次要角色（约 30+ 个）
- 理由：确保角色面板能正确显示所有角色

**决策 3: 文本处理**
- 保留原始英文文本，不进行翻译
- 保留原始标点符号和格式
- 理由：保持公有领域作品的原始性

## Risks / Trade-offs

**风险 1: 文件过大导致编辑器性能问题**
- 缓解措施：这正是测试目的，如果发现性能问题可以后续优化

**风险 2: 手动转换可能遗漏内容**
- 缓解措施：使用自动化工具辅助，人工审核关键场景

**风险 3: 角色名称不一致**
- 缓解措施：统一使用全大写格式（如 "HAMLET", "CLAUDIUS"）

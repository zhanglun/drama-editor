## Why

剧本创作目前缺乏一个简单易用的在线工具。专业编剧、业余爱好者、学生和短视频创作者需要一个现代化的、基于 Web 的解决方案来创作和编辑各类剧本（电影、电视剧、短视频、舞台剧、广告等）。现有工具要么功能过于复杂，要么缺乏协作能力，要么价格昂贵。现在是时候构建一个轻量级但功能完整的剧本编辑器 MVP。

## What Changes

- **新增**基于 Web 的剧本编辑器，支持场景、角色、对白的结构化编辑
- **新增**剧本导出功能，支持标准格式导出（PDF、纯文本等）
- **新增**剧本版本差异对比功能（diff viewer）
- **新增**剧本管理功能，包括创建、保存、加载剧本
- **新增**用户界面，提供直观的编辑体验

## Capabilities

### New Capabilities

- `script-editor`: 核心剧本编辑器，支持场景、角色、对白的结构化编辑，提供所见即所得的编辑体验
- `script-export`: 剧本导出功能，支持导出为 PDF、纯文本等标准格式
- `diff-viewer`: 版本差异对比功能，查看剧本修改历史和变更内容
- `script-management`: 剧本管理功能，包括创建新剧本、保存、加载、删除等操作

## Impact

- **新增代码模块**:
  - 编辑器核心组件（基于富文本编辑器或自定义编辑器）
  - 导出引擎（PDF 生成、文本格式化）
  - Diff 算法和可视化组件
  - 剧本数据模型和存储层
  - 用户界面组件（工具栏、侧边栏、主编辑区）

- **技术栈影响**:
  - 前端框架选择（React/Vue/Svelte）
  - 富文本编辑器库集成（如 Slate.js, ProseMirror, TipTap）
  - PDF 生成库（如 jsPDF, React-PDF）
  - Diff 算法库（如 diff-match-patch, jsdiff）
  - 状态管理方案（如 Redux, Zustand, Jotai）
  - 本地存储或后端 API 集成

- **用户体验**:
  - 直观的编辑界面
  - 快速的导出流程
  - 清晰的版本对比视图
  - 响应式设计，支持不同屏幕尺寸

- **MVP 范围限制**:
  - 暂不包含多人实时协作
  - 暂不包含角色档案和关系图谱
  - 暂不包含故事板和时间线可视化
  - 暂不包含 Final Draft 格式支持（可在后续版本添加）
